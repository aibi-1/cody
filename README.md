# Codify School Bot

Чат-бот школы Codify: отвечает на вопросы родителей и учеников только на основе данных школы.

## Структура

```
school-bot/
├── index.html          ← чат-интерфейс
├── style.css           ← стили (чат-пузыри, мобильная версия)
├── app.js              ← логика чата, рендер, localStorage
├── ai.js                ← запрос к /api/chat
├── school-data.js      ← данные школы
├── prompts.js          ← системный промпт
├── config.js            ← нестроковые настройки (без ключа!)
├── api/
│   └── chat.js          ← Vercel Serverless Function (хранит API-ключ)
└── vercel.json
```

## Почему API-ключ не в config.js

Изначальный план хранил ключ в клиентском `config.js`. Это небезопасно:
любой посетитель сайта может открыть DevTools → Network/Sources и
увидеть ключ в открытом виде, после чего использовать его от вашего имени.

Поэтому ключ переехал на сервер: в переменную окружения Vercel
`OPENAI_API_KEY`, которую читает `api/chat.js`. Браузер обращается
только к `/api/chat`, а сам ключ туда не попадает.

## Деплой на Vercel

1. Залейте папку `school-bot` в репозиторий на GitHub.
2. На [vercel.com](https://vercel.com) → **Add New Project** → импортируйте репозиторий.
3. В настройках проекта → **Environment Variables** добавьте:
   - `OPENAI_API_KEY` = ваш ключ OpenAI
4. Нажмите **Deploy**.

После деплоя бот будет доступен по адресу `https://<project>.vercel.app`.

## Локальный запуск

```bash
npm i -g vercel
vercel dev
```

Перед этим создайте файл `.env` с переменной `OPENAI_API_KEY=sk-...`
(Vercel CLI подхватит его автоматически).

## Проверка критериев

- ✅ Бот отвечает только на основе `school-data.js`
- ✅ Системный промпт запрещает выдумывать факты
- ✅ Если ответа нет — бот говорит "не знаю" + телефон менеджера
- ✅ Быстрые кнопки: Расписание / Цены / Адрес / Записаться
- ✅ История чата в localStorage
- ✅ Состояние загрузки (три точки)
- ✅ Обработка ошибок сети/API
- ✅ Мобильная версия (full-screen на узких экранах)
